    meta name="description" content="lorem ipsum" ===> This adds the description to the webpage.
    meta name="revised" content="company, dd/mm/yyyy" ===> This adds the last revised version information of the webpage.
    meta name="author" content="Author Name" ===> This adds the name of the author of that webpage to the webpage. This maybe the name of the individual or a corporation
    meta http-equiv="refresh" content="5" ===> This refreshes the page after every 5 sec.
    meta http-equiv="refresh" content="5; url = https://example.com" ===> This will refresh the page after 5 seconds and will redirect you to the url there. It can be used when you have to show a popup banner for 5 seconds and the user does not click anything it will redirect you to main site.
